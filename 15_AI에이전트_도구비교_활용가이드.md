## 2. 도구별 상세 분석

### 🦞 OpenClaw — "24/7 상주하는 AI 비서"

```
특징: 내 컴퓨터에 AI가 '살면서' 메신저로 소통
입력: WhatsApp / Telegram / Slack / Discord 메시지
능력: 이메일 관리, 캘린더, 파일 조작, 웹 브라우징, 코드 실행
핵심: 스킬을 AI가 스스로 만들어 장착 (self-hackable)
비용: 오픈소스 무료 + LLM API 비용 (월 $5~50)
```

**AX 활용**: 컨설턴트 본인의 업무 자동화 + 고객 데모용

---

### 🎨 Dify — "노코드 AI 앱 공장"

```
특징: 드래그 & 드롭으로 AI 챗봇/에이전트를 만드는 플랫폼
입력: 웹 UI에서 시각적 워크플로우 편집
능력: RAG 파이프라인 내장, 프롬프트 관리, 모델 A/B 테스트
핵심: 비개발자도 30분 내에 사내 AI 챗봇 배포 가능
비용: 오픈소스 무료 / 클라우드 월 $59~
```

**AX 활용**: 고객사 파일럿의 **1순위 도구** — 가장 빠르게 "작동하는 AI"를 보여줄 수 있음

---

### ⚡ n8n — "업무 자동화의 지휘자"

```
특징: 400+ SaaS/API를 시각적으로 연결하는 자동화 플랫폼
입력: 시각적 노드 편집기 (+ JavaScript 커스텀 가능)
능력: AI 노드 (LLM 호출, 분류, 요약) + 전통적 자동화
핵심: "이메일 오면 → AI가 분류 → Slack 알림 → DB 저장" 같은 흐름
비용: 오픈소스 무료 / 클라우드 월 $20~
```

**AX 활용**: AI를 기존 업무 시스템에 "보이지 않게" 내재화하는 핵심 도구

---

### 🤖 CrewAI — "AI 에이전트 팀 빌더"

```
특징: Python으로 역할별 에이전트를 설계하고 협업시키는 프레임워크
입력: Python 코드 (역할, 목표, 도구 정의)
능력: 리서처 + 분석가 + 작가 등 에이전트 팀 구성
핵심: "조사 → 분석 → 보고서 작성"을 자동 파이프라인화
비용: 오픈소스 무료
```

**AX 활용**: 복잡한 멀티 에이전트 파일럿 (보고서 자동 생성 등)

---

### 🔬 AutoGen (Microsoft) — "에이전트 대화 엔진"

```
특징: 에이전트끼리 대화하며 문제를 해결하는 구조
입력: Python 코드 (고급)
능력: 에이전트 간 토론, 코드 생성 & 실행, 사람 개입 설계
핵심: 학술 연구 수준의 자율 에이전트 시스템
비용: 오픈소스 무료
```

**AX 활용**: R&D 목적, 실무 적용보다 가능성 탐구에 적합

## 3. AX 컨설팅 시나리오별 도구 조합

### 시나리오 A: "사내 FAQ 챗봇 구축" (가장 흔한 첫 프로젝트)

```
[Dify] 단독 사용
├── 사내 문서 업로드 → RAG 자동 구성
├── 챗봇 UI 자동 생성
├── 웹/Slack/API로 배포
└── 소요 시간: 1~2일

비용: 거의 무료 (오픈소스 + API 비용)
난이도: ⭐ (비개발자 가능)
```

### 시나리오 B: "반복 업무 자동화" (이메일·보고서·데이터 정리)

```
[n8n] 중심 + LLM 연동
├── 트리거: 매일 아침 9시 / 이메일 수신 시 / Slack 명령
├── AI 처리: 이메일 분류, 내용 요약, 우선순위 판단
├── 액션: Slack 알림, 스프레드시트 기록, 자동 회신
└── 소요 시간: 3~5일

비용: n8n 무료 + API 비용
난이도: ⭐⭐ (로우코드)
```

### 시나리오 C: "주간 보고서 자동 생성" (멀티 에이전트)

```
[CrewAI] 또는 [n8n + Dify 조합]

Option 1 - CrewAI (코딩):
├── 데이터 수집 에이전트: CRM/DB에서 주간 데이터 추출
├── 분석 에이전트: 전주 대비 분석, 이슈 도출
├── 작성 에이전트: 분석 결과 → 보고서 초안
└── 소요 시간: 1~2주

Option 2 - n8n + Dify (노코드):
├── n8n: 데이터 수집 & 정제 자동화
├── Dify: 수집된 데이터로 보고서 초안 생성
├── n8n: 완성된 보고서를 이메일/Slack으로 발송
└── 소요 시간: 3~5일

난이도: ⭐⭐~⭐⭐⭐
```

### 시나리오 D: "AX 컨설턴트의 개인 무장" (본인 생산성)

```
[OpenClaw] 중심
├── Telegram으로 일정 관리, 리마인더
├── 이메일 자동 분류 & 요약
├── 고객 미팅 전 자동 브리핑 생성
├── LinkedIn 포스트 초안 작성
├── 경쟁사 뉴스 모니터링 → 매일 아침 요약
└── 소요 시간: 설정 1시간, 이후 24/7 운영

난이도: ⭐⭐
```

## 4. 도구 선택 의사결정 플로차트

```
고객이 "AI 도입하고 싶다"고 하면...
│
├── Q: 코딩 가능한 IT팀이 있는가?
│   │
│   ├── YES → Q: 복잡한 멀티 에이전트가 필요한가?
│   │          ├── YES → CrewAI + LangGraph
│   │          └── NO  → n8n + Dify
│   │
│   └── NO → Q: 무엇을 원하는가?
│             ├── "AI 챗봇/검색" → Dify (단독)
│             ├── "업무 자동화"   → n8n (단독)
│             └── "AI 비서"       → OpenClaw
│
└── Q: 데이터 보안이 최우선인가?
    ├── YES → 모든 도구 자체 호스팅 (온프레미스)
    └── NO  → 클라우드 버전 사용 (빠른 시작)
```
