```
              높은 기술 깊이 (구현 가능)
                      │
         ┌────────────┼────────────┐
         │            │            │
         │  AI 전문   │ ★ 우짜노   │
         │  기업      │ (AX 아키텍트)│
         │            │            │
  전략만 ─┼────────────┼────────────┼─ 전략+실행
  (이론)  │            │            │  (End-to-End)
         │  글로벌    │  대형 SI   │
         │  컨설팅    │            │
         │            │            │
         └────────────┼────────────┘
                      │
              낮은 기술 깊이 (전략만)
```

### 나(우짜노)의 포지션: "작지만 날카로운 전문가"

**대형 SI / 글로벌 컨설팅과의 차이점**:
| 비교 항목 | 대형 플레이어 | 우짜노 (AX 아키텍트) |
| --- | --- | --- |
| **비용** | 수천만~수억 | 수백만 단위 (접근 가능) |
| **고객 규모** | 대기업 전용 | 중소~중견기업 특화 |
| **속도** | 제안서만 수 주 | 1주 내 착수 가능 |
| **커뮤니케이션** | 다단계 보고 체계 | 1:1 직접 소통 |
| **유연성** | 패키지 솔루션 | 맞춤형 설계 |
| **실행력** | 하청 구조 가능 | 직접 구현 가능 |

## 4. 기회 영역 (White Space)

### ① 중소·중견기업 AX 컨설팅
- 대형 SI는 관심 없고, 글로벌 컨설팅은 비용이 맞지 않는 **중소·중견기업 시장**
- 직원 50~500명 규모의 기업이 "AI를 어떻게 시작해야 할지" 모르는 상태

### ② 비기술 경영진을 위한 AX 교육
- 기존 플레이어들은 **기술 구현에만 집중**, 조직 문화·변화 관리는 소홀
- "AX를 쉽게 설명하고 조직 변화를 설계하는" 역할의 공백

### ③ 콘텐츠 기반 리드 생성
- 대형 SI는 콘텐츠 마케팅에 약함 → **지식 콘텐츠로 신뢰를 먼저 구축**
- LinkedIn, 뉴스레터, 웨비나를 통한 인바운드 마케팅

### ④ 실전 기반 1:1 AX 코칭
- PM, 팀장 등 실무 리더 대상의 **소규모 AX 코칭/멘토링** 서비스
- 대형 컨설팅에서 다루지 않는 개인 단위 서비스

## 5. 경쟁 대응 전략 요약

> **"경쟁하지 않는 것이 최고의 전략이다."**

1. **시장을 다르게 정의**: 대기업 AX가 아닌, **중소·중견기업 AX 시장**을 타겟
2. **가치를 다르게 제공**: 기술 구현만이 아닌, **전략 + 구현 + 교육**을 원스톱으로
3. **채널을 다르게 활용**: 영업이 아닌, **콘텐츠와 커뮤니티로 고객이 찾아오게**
4. **비용을 다르게 설계**: 수억이 아닌, **수백만 원 단위의 접근 가능한 가격**
