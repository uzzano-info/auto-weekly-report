## 3. 제안 서비스: AXEL 4단계

### Phase 1: Assess (진단) — 2~3주

| 활동 | 산출물 | 참여 인원 |
| --- | --- | --- |
| AX 성숙도 진단 (25개 항목) | 성숙도 보고서 + 레이더 차트 | 경영진, IT 리더 |
| 3계층 페인포인트 인터뷰 | 페인포인트 맵 | 경영진·팀장·실무자 |
| 데이터 준비도 평가 | 데이터 스코어카드 | IT/데이터 담당 |
| 기술 환경 감사 | 현재 아키텍처 맵 | IT팀 |

### Phase 2: X-ray (설계) — 3~4주

| 활동 | 산출물 | 참여 인원 |
| --- | --- | --- |
| AI 비전 워크숍 | AX 비전 선언문 | 경영진 |
| 유스케이스 우선순위 매트릭스 | Impact × Feasibility 매트릭스 | 전 부서 리더 |
| 3단계 로드맵 설계 | Quick Win / Scale / Transform 로드맵 | 경영진, IT |
| ROI 모델링 | 투자 비용 vs 예상 효과 모델 | 경영진, 재무 |

### Phase 3: Execute (실행) — 8~12주

| 활동 | 산출물 | 참여 인원 |
| --- | --- | --- |
| Quick Win 파일럿 1~3개 | 파일럿 보고서 (Before/After) | 파일럿 대상 팀 |
| AI 시스템 구축 | 작동하는 AI 시스템 (데모 가능) | IT팀 + 컨설턴트 |
| 직원 교육 & 변화 관리 | 교육 자료, 워크숍 결과 | 전 직원 |
| 데이터 파이프라인 구성 | 정제된 데이터 + 파이프라인 | 데이터 담당 |

### Phase 4: Leverage (확산) — 지속적 (옵션)

| 활동 | 산출물 | 참여 인원 |
| --- | --- | --- |
| 파일럿 성과 측정 | ROI 검증 보고서 | 경영진 |
| 다른 부서 확산 전략 | 확산 로드맵 | 전 부서 |
| AI 거버넌스 체계 수립 | 거버넌스 정책 문서 | IT + 법무 |
| 지속적 최적화 | 모니터링 대시보드 | IT팀 |
